# 📡 WiFi Analyzer for ESP32-S3 + TFT Display

A real-time WiFi Analyzer built for the **ESP32-S3** with an **ILI9341 SPI TFT display**, providing an intuitive graphical interface for scanning, monitoring, and analyzing nearby WiFi networks.

This project demonstrates embedded GUI development, WiFi scanning, real-time data visualization, and efficient application architecture on the ESP32-S3 platform.

---

## ✨ Features

- 📶 Real-time WiFi network scanning
- 📊 Signal strength (RSSI) visualization
- 🔒 Security type detection
- 📡 Channel information display
- 🔄 Refresh and rescan functionality
- 🎨 Modern TFT graphical interface
- ⚡ Optimized for ESP32-S3 performance
- 🧩 Modular and maintainable code structure

---

## 🛠 Hardware

- ESP32-S3 Development Board
- 2.8" ILI9341 SPI TFT Display
- (Optional) XPT2046 Touch Controller

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/VYLENT01/WiFiAnalyzer_ESP32-S3_TFT.git
```

### 2. Open with PlatformIO

Open the project in **Visual Studio Code** with the **PlatformIO Extension** installed.

### 3. Build & Upload

Compile and upload the firmware to your ESP32-S3 board.
